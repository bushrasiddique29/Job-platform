# 💼 Job Platform (MERN Stack)

A full-stack **Job Portal Web Application** built using the MERN stack (MongoDB, Express, React, Node.js).  
It connects **job seekers** with **employers**, allowing users to apply for jobs and recruiters to post/manage listings.

---

## 🚀 Features

### 👨‍💻 Job Seeker
- User registration & login (JWT authentication)
- Browse job listings
- View job details
- Apply for jobs
- Upload resume (if implemented)
- Track applied jobs

### 🧑‍💼 Recruiter / Admin
- Post new job openings
- Edit & delete job posts
- View applicants
- Manage job listings

- ## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Axios
-  Redux 
- CSS / Tailwind CSS 

### Backend
- Node.js
- Express.js
- Mongoose
- JWT Authentication
- Bcrypt for password hashing
- Multer (for file uploads)

- 🔐 Authentication System
JWT-based authentication
Secure password hashing using bcrypt
Role-based access (User / Recruiter)

📌 Future Improvements
